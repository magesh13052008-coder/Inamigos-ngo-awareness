# Inamigos-ngo-awareness
InAmigos Foundation awareness webpage built with HTML &amp; CSS, showcasing the NGO's mission and six flagship projects — Seva, Bachpanshala, Jeev, Udaan, Prakriti, and Vikas — covering food security, education, animal welfare, women's empowerment, environment, and skill development. Includes impact stats, certifications, and volunteer/donate CTAs.
